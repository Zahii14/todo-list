# 📝 To-Do List App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add tasks
- pressing Enter add a task
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
